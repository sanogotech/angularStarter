## Install

- NodeJs  :  node --version
- npm  :  npm  --version
- Anglar CLI (ng)
```
npm cache clean --force
npm install -g  @angular/cli
ng version

```
* Anglar Material
```
ng add @angular/material
```

# Create Sample Project with angular/cli

```
npm cache clean --force
ng new  mysampleproject
```
